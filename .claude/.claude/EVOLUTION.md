# Evolution Log

> Harness 的进化历史。每次改进都记录在这里，让系统越用越好。

## 进化层级

| Layer | 触发条件 | 自动化程度 | 需确认 |
|-------|---------|-----------|--------|
| L1 静默记录 | 任何 feedback | 全自动 | 否 |
| L2 规则升级 | 同一条 feedback ≥3 次 | 半自动 | 是 |
| L3 Skill 优化 | Skill feedback 评分持续低 | 半自动 | 是 |
| L4 新 Skill | 模式重复无覆盖 | 半自动 | 是 |
